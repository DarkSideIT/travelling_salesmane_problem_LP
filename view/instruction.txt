В этом примере мы создаем главное окно приложения и добавляем поле ввода (entry) для ввода размерности матрицы n. 
При нажатии на кнопку "Submit" (submit_button) вызывается функция create_matrix_form, которая получает введенное значение n и создает форму для заполнения матрицы.

После ввода размерности n, поле ввода и кнопка "Submit" удаляются, а метка (label) обновляется, указывая на ввод матрицы определенной размерности. 
Затем создаются поля ввода (Entry) для каждого элемента матрицы размерности n x n, и они располагаются на форме с использованием сетки (grid).

После заполнения матрицы и нажатия кнопки "Display Matrix", вызывается функция get_matrix, которая собирает значения из полей ввода и 
передает их в функцию display_matrix для вывода матрицы.

Обратите внимание, что в данном примере матрица выводится в консоль с помощью функции print. Вы можете изменить функцию display_matrix в соответствии с 
вашими потребностями для вывода матриц
